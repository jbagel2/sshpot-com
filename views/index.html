About

SSHPot.com is a open-source web service for receiving data from ssh-pot daemons, which act as a honeypot for ssh login attempts and records all user/password combinations along with the remote address. In order to run the daemon you must configure OpenSSH to run on a different port and let the honeypot run on port 22. We also record commands being run as well as proxy request data.

Login API

GET /api/ssh_logins.json

API to retrieve a list of login attempts, 50 per page.

Params
  • page: a page offset, default is page 1
Example Response 
{
    "page":1,
    "total":1171,
    "data":[
        {
            "id":1174,
            "remote_addr":"183.56.129.146",
            "username":"carlton",
            "password":"carlton",
            "created_at":"2014-07-05T16:32:18.738787-04:00"
        },
        {
            "id":1173,
            "remote_addr":"116.10.191.166",
            "username":"admin",
            "password":"1qaz2wsx!@#",
            "created_at":"2014-07-05T16:27:38.105188-04:00"
        }
    ]
}

Command API

GET /api/ssh_commands.json

API to retrieve a list of commands run, 50 per page.

Params
  • page: a page offset, default is page 1
Example Response 
{
    "page":1,
    "total":31,
    "data":[
        {
            "command": "passwd root123",
            "created_at": "2014-08-16T18:05:13.020015-04:00",
            "id": 2
        },
        {
            "command": "whoami",
            "created_at": "2014-08-16T17:54:27.112847-04:00",
            "id": 1
        }
    ]
}

Proxy API

GET /api/ssh_proxy.json

API to retrieve a list of proxy requests, 50 per page.

Params
  • page: a page offset, default is page 1
Example Response 
{
    "page":1,
    "total":31,
    "data":[
        {
            "data": {
                "Accept": "*/*",
                "Host": "google.com",
                "Method": "GET",
                "URI": "/?q=foobar",
                "User-Agent": "curl/7.30.0"
            },
            "created_at": "2014-08-17T19:10:49.386791-04:00",
            "id": 10
        }
    ]
}
